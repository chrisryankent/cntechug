Tips and Tricks
  • HTML tricks: You can automate website designing using ChatGPT and other simple open AI bots and instruct it to generate code given known parameters.
  • CMD tricks: You can change your prompt color. In Windows, open CMD and type color --help. Syntax will be shown, e.g., color 1 for blue.
  • Hack tricks: For computers on a network, you can shut down all of them by just one command. In CMD type: shutdown all.
